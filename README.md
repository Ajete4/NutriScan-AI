# 🧠 NutriScan AI

NutriScan AI është një web aplikacion inteligjent për analizën e ushqimit dhe gjenerimin e planeve ushqimore të personalizuara duke përdorur AI.

Aplikacioni u lejon përdoruesve të:
- 📸 skanojnë ushqimin me foto
- 🤖 marrin analizë automatike (kalori, proteina, karbohidrate, yndyra)
- 🥗 gjenerojnë plane ushqimore ditore, javore dhe mujore
- 💾 ruajnë dhe menaxhojnë planet e tyre

---

# 🚀 Features kryesore

## 🤖 AI Nutrition Plans
- Gjenerim automatik i planeve ushqimore
- Daily / Weekly / Monthly structure
- Ruajtje dhe histori e planeve

## 📸 Meal Scanner
- Upload image
- AI analysis (calories & macros)
- Save meal në database

## 📊 Dashboard
- Progress tracking
- Plan history
- Health insights

---

# 🛠️ Teknologjitë

- **Frontend:** Next.js (App Router), React, TypeScript  
- **Styling:** Tailwind CSS  
- **Animations:** Framer Motion  
- **Icons:** Lucide React  
- **Backend:** Next.js API routes  
- **Database:** Supabase (PostgreSQL)  
- **Storage:** Supabase Storage  
- **AI Integration:** API për analizë ushqimi
